This is a Java program to work with graphs. It builds a graph using an Adjacency List. It can run BFS and DFS traversals, and I also added Dijkstra's algorithm for the 5% bonus task to find the shortest paths in a weighted graph.File structureVertex.java — holds the node id.Edge.java — connects nodes and stores weights.Graph.java — has the adjacency list, weight matrix, and all algorithm logic.Experiment.java — creates 10, 30, and 100 vertices graphs to measure execution time.Main.java — just runs the tests.Algorithms implemented1. BFS (Breadth-First Search)Goes through the graph level by level. It uses a standard Java Queue to keep track of vertices to visit next.2. DFS (Depth-First Search)Goes deep into the graph branches first. It is implemented with a recursive helper method dfsHelper and a visited[] array.3. Dijkstra's Algorithm (Bonus Task)Finds the shortest paths from node 1 to all other nodes.I changed the graph to be weighted. The edges get random weights from 1 to 10 in the experiments.As allowed in the bonus pdf, I did not use a PriorityQueue. It uses standard arrays for distances and tracking visited nodes, and simple for loops to find the minimum distance vertex.My ResultsGraph Structure OutputWhen the program starts, it prints how the vertices are connected and shows weights like neighbor(w:weight).
